# 🧰 研发七部工具包

轴承数据分析与报告生成桌面工具集。

## 📦 功能模块

### 📊 边缘应力分析

对轴承 HTM 仿真数据进行解析、异常峰值去除、应力曲线绘制和报告生成。

- 📂 **数据导入** — 选择文件夹，递归扫描 HTM 文件，支持多次追加与去重
- 🔍 **峰值去除** — 基于阈值自动检测并去除异常峰值
- 📈 **报告生成** — 一键生成应力曲线 PNG（中/英文）、载荷与应力雷达图、最大应力汇总 Excel
- ✏️ **交互式预览** — 独立窗口查看 Plotly 图表，支持拖拽编辑数据点实时更新报告
- 🎯 **雷达图调整** — 自定义载荷/应力雷达图的径向最小值

### ⚙️ 设置

- 🌗 **外观切换** — 浅色 / 深色 / 跟随系统三种模式
- 🌐 **语言** — 预留中文/英文切换（即将推出）
- 💾 **持久化** — 设置数据通过 electron-store 保存在本地 JSON 文件中，重启后自动恢复

## 🚀 使用步骤

1. 启动应用，在侧边栏点击「边缘应力」进入分析页面
2. 点击「选择文件夹」导入包含 HTM 文件的目录
3. 根据需要调整峰值阈值和图片尺寸参数
4. 点击「开始解析」，等待进度条完成
5. 解析完成后，点击文件名查看报告预览
6. 在预览窗口中可切换滚道、列号，拖拽编辑数据点
7. 生成的报告文件保存在输出目录的 `res-output/` 子目录中

## 📁 输出文件说明

解析完成后，在输出目录下生成以下文件：

```
res-output/
└── {组名}/
    ├── {文件名}-内滚道接触应力-第{N}列-中.png   # 内圈应力曲线（中文）
    ├── {文件名}-内滚道接触应力-第{N}列-英.png   # 内圈应力曲线（英文）
    ├── {文件名}-外滚道接触应力-第{N}列-中.png   # 外圈应力曲线（中文）
    ├── {文件名}-外滚道接触应力-第{N}列-英.png   # 外圈应力曲线（英文）
    ├── {文件名}-第{N}列-载荷雷达图.png          # 载荷分布雷达图
    ├── {文件名}-第{N}列-应力雷达图.png          # 应力分布雷达图
    └── 最大应力汇总.xlsx                        # 所有文件所有列的最大应力值
```

## 🔧 技术栈

- **前端** — Electron + Vue 3 + TypeScript + Element Plus
- **后端** — Python (FastAPI)
- **构建** — electron-vite + PyInstaller

## 💻 开发

```bash
# 前端
cd frontend
cnpm install     # 安装依赖
npm run dev      # 启动开发服务器

# 后端
cd backend
uv sync          # 安装依赖
uv run main.py   # 启动 FastAPI 服务 (端口 8000)
```
